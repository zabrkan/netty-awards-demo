The runbook — issuing a ticket

  1. A request lands below with the donor’s name and their receipt number — the private number Facebook emails only the person who donated.
  2. Open the fundraiser’s donor list and confirm the donation: St. Jude fundraiser ↗. As organizers we see every donor’s name and amount; the receipt # is the proof it’s really them (and covers anonymous gifts, where the name won’t show on the list).
  3. Match found → Approve & issue. A single-use code is generated and their ballot unlocks automatically on their device. No match → Reject (they’re told to double-check their receipt).
  4. Optional DM — send the congrats + code with the template below (the code matters only if they vote on a different device than the one they requested from).
